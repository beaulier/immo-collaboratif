Les pré-requis nécessaires au fonctionnement du widget se trouvent dans le dossier courant.
---------------------------------------------------------------------------------------------------------------------------
>Installez WampServer et placez l'ensemble des dossiers du projet dans le dossier c:\wamp\www\
>Rendez vous sur "127.0.0.1/PhpMyAdmin" et importez la base de donnees à partir du fichier database.sql
>Lancer une invite de commande shell
>Entrez la commande "cd C:\wamp\www\tchat" pour vous positionner dans le bon dossier
>Installez le module nodejs grace à la commande "npm install"
>Lancez le serveur avec la commande  "node server3.js"
---------------------------------------------------------------------------------------------------------------------------
Note : la database.sql est une base de test contenant de fausses données
---------------------------------------------------------------------------------------------------------------------------
Rendez vous maintenant à l'adresse "http://127.0.0.1/projet-immo/php"
